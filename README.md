# 🧚‍♀️Bubble CLI

A very small CLI script using go-lang and Bubble Tea from Charm CLI.

### 📋Aims of App

- User is able to input their name
- The user is able to see the following message within the CLI, "Hello, {name-input}"

### 👩‍💻Technical Details

- go-lang
- Bubble Tea

### 🔧How to Run the App

```bash
  go run main.go
```
